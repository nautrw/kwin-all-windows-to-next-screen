# kwin-all-windows-to-next-screen
A Kwin script that moves all windows to their next screen.
They are moved independently, so it can be used to swap the windows' screens.
## Installation
To install, run the following commands:
```sh
git clone https://github.com/nautrw/kwin-all-windows-to-next-screen.git
cd kwin-all-windows-to-next-screen
kpackagetool6 --type KWin/Script -i .
```
Then log out and back in to KDE in order to apply the changes.
## Usage
After the script is enabled, there will be a shortcut called "Move All Windows To Next Screen."
You must set a keybind for it.
